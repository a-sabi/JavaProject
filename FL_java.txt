ОСНОВНЫЕ ФУНКЦИИ
1)Проигрывание песен в формате MP3
2)Возможность поиска песен из всего списка песен или из плейлиста
3)Перемешивание песен 
4)Возможность создавать новый плейлист
5)Перемотка песни с помощью бегунка
6)Возможность переключать, останавливать трек

ДОПОЛНИТЕЛЬНЫЕ ФУНКЦИИ
1)Видеоклипы к некоторым песням, которые будут показываться на фоне плеера
2)Возможность отправки данных пользователем при возникновении проблем 
3)Раздел с текстом для каждой песни
